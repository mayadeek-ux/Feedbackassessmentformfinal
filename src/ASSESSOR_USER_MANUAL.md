### Conducting an Assessment

#### Step 1: Start the Assessment
1. Click **"Start Assessment"** on a candidate card
2. The Individual Scoring Page will open

#### Step 2: Navigate the Interface

**Left Navigation Panel:**
- Shows all 10 criteria
- Click any criterion to jump to it
- Progress indicators show completion status
- Completed criteria are highlighted

**Main Scoring Area:**
- Displays the selected criterion and its 10 sub-criteria
- Each sub-criterion has a **checkbox**
- Check boxes to award points (1 point per tick)
- **Criterion score** updates in real-time as you tick boxes

**Right Summary Panel:**
- **Overall Score:** Running total out of 100
- **Performance Band:**
  - üî¥ Limited (0-39): Requires significant development
  - üü° Developing (40-59): Growing capabilities
  - üîµ Strong (60-79): Solid performance
  - üü¢ Exceptional (80-100): Outstanding performance
- **Criteria Breakdown:** Shows score for each criterion
- **Progress Bar:** Visual completion indicator

#### Step 3: Scoring Tips

**Objective Assessment:**
- Tick boxes based on **observable behaviors** during the session
- Don't tick if you didn't witness the behavior/capability
- It's okay to have low scores‚Äîhonesty helps development

**Taking Notes:**
- Use the **Notes/Comments** field at the bottom
- Document specific examples of behaviors
- Note any red flags or exceptional moments
- These notes appear in the generated reports

**Auto-Save:**
- Your progress saves automatically every few seconds
- You'll see a "Saved" indicator
- If disconnected, reconnect and your progress will be preserved

#### Step 4: Submit Assessment
1. Review your scoring across all 10 criteria
2. Add final notes or summary comments
3. Click **"Submit Assessment"**
4. Confirm submission (this cannot be undone)
5. You'll be redirected to the dashboard

## Scoring Methodology

### The Tick-Box System

**Philosophy:**
- Simple, objective, and fast
- Each behavior/capability is binary: observed or not observed
- Reduces subjective bias
- Easy to explain and defend

**How It Works:**
1. Read each sub-criterion carefully
2. Ask yourself: "Did I observe this during the assessment?"
3. If YES: Tick the box ‚úÖ
4. If NO or UNCLEAR: Leave unticked ‚ùå
5. Scores accumulate automatically

### Performance Bands

**Understanding the Bands:**

| Score Range | Band | Meaning | Development Focus |
|-------------|------|---------|-------------------|
| **80-100** | Exceptional üü¢ | Outstanding performance, exceeds expectations | Leadership roles, mentoring others |
| **60-79** | Strong üîµ | Solid competence, meets expectations | Refinement, advanced skills |
| **40-59** | Developing üü° | Growing capabilities, has potential | Targeted training, practice opportunities |
| **0-39** | Limited üî¥ | Foundation building needed | Intensive support, basic skill development |

### Calibration Tips

**Consistency Across Assessments:**
- Use the same standards for all candidates
- Don't inflate scores to "be nice"‚Äîhonest feedback helps development
- If uncertain about a sub-criterion, leave it unticked
- Consult with fellow assessors if you need calibration

**Avoiding Biases:**
- **Halo Effect:** Don't let one strong area inflate all scores
- **Recency Bias:** Consider the entire session, not just the end
- **Contrast Effect:** Don't compare candidates to each other
- **First Impression:** Give candidates the full assessment period

## Troubleshooting

### Common Issues & Solutions

#### Login Problems

**Issue:** Not receiving magic link email
- **Solution:**
  - Check spam/junk folder
  - Verify email address is correct
  - Wait 2 minutes and try again
  - Contact admin if email still doesn't arrive

**Issue:** Magic link expired
- **Solution:**
  - Request a new magic link
  - Magic links are valid for 15 minutes only
  - Use the link immediately after receiving it

#### Assessment Issues

**Issue:** Assessment not saving
- **Solution:**
  - Check connection status indicator (top right)
  - Ensure internet is stable
  - Refresh page‚Äîauto-save should restore progress
  - Contact admin if data is lost

**Issue:** Can't submit assessment
- **Solution:**
  - Ensure all required fields are completed
  - Check for error messages
  - Try refreshing the page
  - Contact admin if problem persists

**Issue:** Scores not calculating correctly
- **Solution:**
  - Refresh the page
  - Re-check ticked boxes
  - Each individual criterion tick = 1 point (max 10 per criterion)
  - Each group criterion tick = 2 points (max 20 per criterion)

#### Platform Navigation

**Issue:** Dashboard not showing assignments
- **Solution:**
  - Check event switcher‚Äîyou may be on wrong event
  - Refresh page
  - Log out and log back in
  - Contact admin to verify assignments

**Issue:** Real-time updates not working
- **Solution:**
  - Check connection status (green = connected)
  - Refresh page to reconnect
  - Check internet connection
  - Contact admin if issue continues
